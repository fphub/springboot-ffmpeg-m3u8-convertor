# springboot-ffmpeg-m3u8-convertor

#### Description
springboot+ffmpeg，将视频转换为 m3u8 格式。支持 .mp4 | .flv | .avi | .mov | .wmv | .wav 格式视频转换。指定文件路径 + 文件上传转换两种转换方式。
